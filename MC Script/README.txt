# Harpoon_maker

usage: Harpoon_maker [-h] [-p PROCESSES] [-s SAMPLES] [-o OUTPUT] [-l LOWER_NODE] [-u UPPER_NODE]

This is a program that will output csv data files about drawn Monte Carlo network samples.

Optional Arguments:

-h, --help <br/>
show this help message and exit

-p PROCESSES, --processes PROCESSES<br/>
Set number of parallel process to run.

-s SAMPLES, --samples SAMPLES<br/>
Set total number of Monte Carlo samples to run.

-o OUTPUT, --output OUTPUT<br/>
Set the path to store the output folder.

-l LOWER_NODE, --lower_node LOWER_NODE<br/>
Set the inclusive lower node limit that the MC sampling will start from.

-u UPPER_NODE, --upper_node UPPER_NODE<br/>
Set the inclusive upper node limit that the MC sampling will run up to.
